# 📈 Stock Portfolio Tracker

## Overview

This project was developed as part of the **CodeAlpha Python Programming Internship**.

The Stock Portfolio Tracker is a Python application that allows users to enter stock names and quantities, then calculates the total investment value using predefined stock prices.

## Features

* Accepts user input for stock names and quantities.
* Uses a predefined dictionary of stock prices.
* Calculates the total investment value.
* Optionally saves the results to a text or CSV file.

## Technologies Used

* Python 3
* Visual Studio Code

## How to Run

1. Clone or download this repository.
2. Open the project in Visual Studio Code.
3. Run the following command:

```bash
python stock_tracker.py
```

4. Enter the required information when prompted.
